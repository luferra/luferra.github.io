WEBSITE, WORKSHOPS, WIREFRAME, DESIGN

Unimi.it

Redesign Unimi.it and 135+ course websites to improve accessibility, multilingual support, and student engagement.


ENVIRONMENT

University of Milan

DATE

January 2018 - May 2019

ROLE

UX Researcher and UX Designer

Phase 1

UX Research & Strategy

Led a university-wide digital transformation through evidence-based design:

  • Workshop Facilitation: Coordinated cross-departmental UX workshops to redefine content hierarchies for faculty landing pages and course portals.
  • User Testing:
    • Treejack Tests: Validated navigation structures for intuitive information access.
    • First-Click Tests: Optimized key user flows (e.g., enrollment, course discovery).
  • Card Sorting: Restructured 1,000+ content items with input from students, faculty, and staff.
  • Accessibility Audits: Ensured WCAG compliance for bilingual (Italian/English) content.

Phase 2

UX Design & Implementation

  • Mobile-first layouts (tested across devices).
  • Consistent UI components (cards, accordions, tabs).
  • Streamlined faculty page transitions from legacy systems.
  • Tested critical pages (card sorting, first-click tests, treejack tests, questionnaires)

Phase 3

Outcome & Ongoing Impact

Launched 2019: Unified portal serving 60,000+ users with:

  • 100% bilingual access.
  • Enhanced accessibility features (screen reader compatibility, contrast ratios).

Post-Launch Leadership:

  • Trained 200+ staff on CMS usage.
  • Established continuous feedback loops via quarterly UX assessments.
