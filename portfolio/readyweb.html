Unimi ReadyWeb: Managed WordPress Platform for Academic Sites

Project Date January, 2023
Tools Figma, WordPress, WCAG Testing.
Environment University of Milan
Artifacts ReadyWeb Websites list (in talian only)

Project Goal

Create a no-code WordPress solution for University of Milan researchers to build compliant, mobile-ready websites without technical overhead.

UX & System Design Leadership

End-to-End Platform Design:

  • Developed a Figma component library (headers, footers, color system) to standardize 50+ research group/event sites.
  • Designed intuitive CMS templates with customizable branding (logos, 6-color palette).
  • Accessibility Compliance: Ensured WCAG adherence across themes, managed by ICT (cookie banners, screen reader optimization).

Key Features & Process

User-Centric Development:

  • Simplified editing interface for non-technical users (drag-and-drop modules).
  • Implemented advanced post filters (categories, authors, dates) through iterative Figma prototyping.

Scalable Maintenance

Partnered with ICT to automate updates, security patches, and bug fixes.

  • Reduced user training needs with guided onboarding flows.

Outcome & Impact

Launched 2023: 100+ research sites deployed with ReadyWeb.

  • Metrics:
    • 80% reduction in site setup time for researchers.
    • 100% compliance with EU privacy/university accessibility standards.
  • Continuous Improvement: Quarterly UX reviews to add features (e.g., location-based filtering).

Skills Demonstrated:

  • UX Systems Design
  • Component Libraries
  • Stakeholder Collaboration
