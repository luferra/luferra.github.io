1. DooBeDoo: Child Task Manager Tangible Interface

Product Design | Child-Computer Interaction | Jan 2025

Core Concept

Interactive cube helping children (4-10) track after-school tasks through physical tokens. Parents and kids jointly select 4 daily tasks represented by magnetic badges.

Key Features

  • Coin-slot mechanism provides tactile feedback (LED turns green)
  • Earned "time points" displayed via top-mounted timer
  • Customizable task badges (homework, teeth brushing, etc.)

Development Process

  • Designed bee mascot logo (symbolizing diligence)
  • Prototyped wooden cube structure with:
    • 4 task faces
    • Token insertion slots
    • LED feedback system
  • Created playful brand identity with "Hoss Round" font

Output: Functional physical prototype

3. Sony TC-150 Tape Player Vintage Redesign

Product Redesign | Hybrid Fabrication | Dec 2024

Design Approach

Faithful recreation of 1970s portable recorder

Modernized with:

  • Laser-cut wooden casing
  • Functional buttons (play, skip, volume)
  • Tape compartment door mechanism

Key Components

  • Spring-loaded lever system
  • Multi-layer laser-cut construction
  • Magnetic closure system

Process

  • Paper sketching → Cardboard prototype → Fusion 360 model
  • Solved mechanical challenges (lever mechanism, volume control)
